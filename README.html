C Tutorial

This example the same source code and test cases that are referenced in the C language tutorial in the Getting Started User Guide.

Concepts Covered:

Basic Test Case Construction

Basic test case construction is covered, including input and expected values for scalar and structure types. The test cases that you build as part of the Getting Started User Guide are named PLACE_ORDER.001 and PLACE_ORDER.002.

Importing Test Data from a CSV File

To see how easy it is to import data from a CSV file, just right-click on the test node "(MAP)CSV_MAP_EXAMPLE" and choose "Create Tests." The "source data" of the test case data is the file $VECTORCAST_DIR/tutorial/c/csv_test_data.csv.
